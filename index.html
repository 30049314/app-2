How to Use

Stopwatch: To start the stopwatch, click the Start button. Click the button again to pause it. To reset the time to zero, press the Reset button.

Lap Timer: Click Start to begin timing. Press Lap to record a lap time while the timer is running. To stop the timer, click Start again. Use Reset to clear the timer and lap list.

Alarm: Set your desired alarm time using the Set Alarm button. When the alarm time is reached, an alert sound will play. To cancel the alarm, press the Cancel Alarm button.
